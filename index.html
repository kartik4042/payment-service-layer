05 — Fault Tolerance

Circuit Breaker Pattern

⊙
CLOSED
Normal operation. All requests flow through to the provider. Failures are counted but do not yet exceed the threshold. This is the healthy state.
✕
OPEN
Provider isolated. All requests immediately fail-fast without attempting the provider. Triggers automatic fallback routing. Duration: configurable wait period.
◑
HALF-OPEN
Probe mode. A single test request is allowed through. If it succeeds, the circuit closes (recovery). If it fails, the circuit reopens (extended isolation).
